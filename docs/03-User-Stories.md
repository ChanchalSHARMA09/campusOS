# User Stories - CampusOS

## Overview

This document contains the user stories for CampusOS. Each user story follows the standard Agile format:

> **As a [user], I want [feature] so that [benefit].**

---

# Student User Stories

### Authentication

- As a student, I want to register using my college email so that I can access the platform.
- As a student, I want to securely log in so that only I can access my account.
- As a student, I want to reset my password if I forget it.

### Dashboard

- As a student, I want to view my dashboard so that I can quickly access all important information.
- As a student, I want to update my profile details.

### Attendance

- As a student, I want to view my attendance percentage so that I can monitor my academic progress.
- As a student, I want to receive attendance shortage alerts.

### Assignments

- As a student, I want to view assignments uploaded by faculty.
- As a student, I want to submit assignments online before the deadline.
- As a student, I want to see assignment deadlines.

### Notes

- As a student, I want to download lecture notes.
- As a student, I want to organize notes by subject.

### Placement

- As a student, I want to browse internships and job opportunities.
- As a student, I want to apply for placement drives.
- As a student, I want to create and update my resume.
- As a student, I want to practice mock interviews.

### Community

- As a student, I want to create posts.
- As a student, I want to like and comment on posts.
- As a student, I want to follow discussions.

### Chat

- As a student, I want to send messages to classmates.
- As a student, I want to create group chats.
- As a student, I want to receive instant notifications.

### Events

- As a student, I want to register for campus events.
- As a student, I want to receive event reminders.
